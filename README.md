
# 🚀 Routine Pro – Smart Teacher Management System

## 📌 Overview
Routine Pro is a web-based application designed to help teachers manage lecture schedules, track student attendance, and maintain daily wellness logs in one unified platform.

---

## 🎯 Features

### 📅 Lecture Scheduling
- Add and manage lecture tasks
- Set deadlines and priorities
- Track pending tasks
---
### 👨‍🏫 Attendance Management
- Add students batch-wise
- Mark daily attendance
- Store attendance records
---
### 📊 Attendance Analytics
- Calculate attendance percentage
- Visual progress indicators
---
### 🌱 Wellness Tracking
- Record mood and stress level
- Maintain daily journal
- View past logs

---

## 🛠️ Tech Stack

- **Backend:** Python (Flask)
- **Database:** SQLite (SQLAlchemy)
- **Frontend:** HTML, CSS, Bootstrap
